FAQ

[1] Uiscale issue
If you set the uiscale to a really high scale, here's what to do.
  1. Download the application named Zarchiver. (You must allow full file access to Zarchiver and FluxusZ)
  2. Open the folder named FluxusZ.
  3. Open the folder named Workspace inside the FluxusZ folder.
  4. Find/search for the file named uiscale.txt.
  5. Open it and edit the value to a lower number (example: 1.3 or 1.5).
  6. Save the file and return to Roblox.
[2] Claiming premium key while a key is active
Follow these steps to replace an active key:
  1. Download the application named Zarchiver. (You must allow full file access to Zarchiver and FluxusZ)
  2. Open the folder named FluxusZ.
  3. Open the folder named Workspace inside the FluxusZ folder.
  4. Find/search for the file named FluxusZ Key.lua.
  5. Rename the extension to .txt (example: FluxusZ Key.txt).
  6. Open the file, remove the current key and replace it with your key / the key given to you.
  7. Save, rename the extension back to .lua, and return to Roblox.
